================================
HOLLOWINK HORROR BLOG - SETUP
================================

QUICK START:
-----------

1. Extract the horror-blog-react folder

2. Open terminal in the horror-blog-react folder

3. Install dependencies:
   npm install

4. Start the development server:
   npm start

5. Open http://localhost:3000 in your browser

That's it! The site should now be running locally.

CUSTOMIZATION:
-------------

1. Add Your Stories:
   - Edit src/data/stories.js
   - Add new story objects with your content

2. Update Author Info:
   - Edit src/pages/About.jsx
   - Change "Your Name" and bio text

3. Modify Colors:
   - Edit tailwind.config.js
   - Change color values in theme.extend.colors


BUILD FOR PRODUCTION:
--------------------

When ready to deploy:
npm run build

This creates a 'build' folder with optimized files ready for hosting.


TROUBLESHOOTING:
---------------

If you get errors:
1. Make sure Node.js is installed (v14 or higher)
2. Delete node_modules folder and package-lock.json
3. Run npm install again
4. Clear browser cache


TECH STACK:
-----------

- React Router v6
